HTML Cheat Sheet

This cheat sheet includes some commonly used HTML tags and attributes.

Basic Tags

  • <!DOCTYPE html> - Defines the document type.
  • <html></html> - The root element of an HTML page.
  • <head></head> - Contains metadata, links, and title.
  • <body></body> - The content of the HTML page.

Text Formatting Tags

  • <h1> to <h6> - Heading tags from largest (h1) to smallest (h6).
  • <p> - Defines a paragraph.
  • <strong> - Makes text bold.
  • <em> - Italicizes text.
  • <br> - Inserts a line break.
  • <hr> - Inserts a horizontal line.

Links and Images

  • <a href="url">Link Text</a> - Creates a hyperlink.
  • <img src="path/to/image" alt="description"> - Displays an image.

Lists

  • <ul><li>Item</li></ul> - Unordered list.
  • <ol><li>Item</li></ol> - Ordered list.

Tables

  • <table> - Defines a table.
  • <tr> - Table row.
  • <th> - Table header.
  • <td> - Table data.

Forms

  • <form></form> - Defines a form.
  • <input type="text"> - Single-line text input.
  • <textarea></textarea> - Multi-line text input.
  • <button>Click Me</button> - Button element.

Media

  • <audio controls><source src="path/to/audio" type="audio/mpeg"></audio> - Embeds audio.
  • <video controls><source src="path/to/video" type="video/mp4"></video> - Embeds video.
